# html
- 首先是button的用法
    - 里面用@click来指定事件
        - 指定的事件用双引号包裹
        - 指定的事件可以带参数，括号即可
        - <!-- button @click指定事件，可以带参数，事件用function来定义 -->
    - 事件用function定义在script里面
    
# css
- display
    - 控制一个div内部多个元素怎么排布
    - flex：自定义规则，方便Justify-content,align-items
    - block：一人一排
    - inline：行内
- justify-content
    - space-between：把东西放两边
    - flex-start:把东西都放左边
    - center：放中间
    - flex-end：放右边
# ts/js
- 判断类型常用typeof，感觉对于ts这种类型灵活的要经常用，结果来对比字符串
    - typeof 对象 === 'string' 来判断是否是string
- 非空检查的常用：直接if判断 ！xxx.value
- 引用的时候，引用类型是import type，引用函数好像是import
# vue
- 组件和组件的引用
    1. 目录：src/components
    2. 通常原生数据用ref响应式，但是要演变的派生数据就用computed比较合适
    3. computed的内容也要value调用别忘了
- 初始化的时候的动作
    1. 直接对onMounted函数动手
- 抽象：契约进行规定的时候：class,type,interface都可以进行规定
    - class：带有方法
    - interface: 只有模型，并且没有联合类型
    - type: 有联合类型
